This Privacy Policy describes how CosmiX BOT (“Bot”, “we”, “us”, or “our”) processes Discord-related information when the Bot is added to or used in a Discord server.

By using the Bot, you agree to the collection and use of information in accordance with this policy and Discord’s own Terms of Service and Privacy Policy.

2. Information We May Collect

Depending on the features enabled by a server, CosmiX BOT may process the following categories of data:

  • Discord user IDs — to identify users and apply bot features.
  • Discord server (guild) IDs — to configure server-specific settings.
  • Channel IDs / role IDs — to manage roles, commands, and logs.
  • Voice activity or activity time — if a feature tracks time spent in voice channels.
  • Messages or command content — only when required for a feature you enable.
  • Logs and usage data — to help improve reliability and troubleshoot issues.

We do not intentionally collect sensitive personal information unless you explicitly provide it through a feature that requires it.

3. How We Use Data

We use the collected data only for legitimate bot functionality, including:

  • automating server tasks
  • managing temporary roles and permissions
  • tracking feature usage such as VC time
  • responding to slash commands and interactions
  • debugging, monitoring, and improving performance

4. Data Sharing

We do not sell your data. We do not trade it like some shady side quest merchant. Data may be shared only in the following limited cases:

  • with Discord, as required for the Bot to operate through Discord’s API
  • with trusted hosting or infrastructure providers that help run the Bot
  • when required by law, legal process, or to protect the Bot and its users

5. Data Retention

Data is stored only for as long as it is needed to provide the Bot’s features or maintain server settings.

When data is no longer needed, we may delete it automatically or upon a valid deletion request, subject to technical limitations and legal obligations.

6. Security

We take reasonable steps to protect stored data from unauthorized access, misuse, or disclosure. No system is 100% secure, so we cannot guarantee absolute security.

Server owners should also manage permissions carefully and avoid granting unnecessary access to the Bot.

7. Your Choices and Rights

Depending on the server and the features enabled, you or the server owner may request access to, correction of, or deletion of data associated with the Bot.

To make a request, contact the support info listed below. We may need to verify the request before taking action.

8. Children’s Privacy

CosmiX BOT is intended for users who are allowed to use Discord under Discord’s Terms and applicable law. If you believe data has been collected from a minor in violation of the law, contact us immediately.
